# IPO Apply

A tiny native Kotlin Android app for saving MeroShare IPO apply account setup.

## Run

Open this folder in Android Studio, let Gradle sync, then run the `app` configuration on an emulator or device.

## Build APK

```powershell
.\gradlew.bat assembleRelease
```

The tiny release APK is generated at `app/build/outputs/apk/release/app-release.apk`.
